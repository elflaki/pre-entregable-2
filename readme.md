# Página Web de Servicios Médicos de Emergencia

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de una página web para una empresa de servicios médicos de emergencia. La página incluye información sobre los servicios ofrecidos, el equipo médico, y cómo contactar con la empresa.

## Estilos Utilizados

Se han utilizado estilos CSS para dar formato y diseño a la página web. A continuación, se detallan algunos de los estilos más relevantes:

- **Estilos de la Barra de Navegación:** Se ha aplicado un fondo oscuro con borde blanco para resaltar la barra de navegación.
- **Títulos:** Los títulos principales y secundarios se han diseñado con una fuente Arial y colores contrastantes para mejorar la legibilidad.
- **Secciones de la Página:** Cada sección de la página se ha diseñado con un borde sutil y un fondo que contrasta con el contenido.
- **Imágenes:** Se han aplicado estilos para que las imágenes se ajusten correctamente al contenedor, con bordes redondeados y sombras suaves.
- **Formulario de Contacto:** El formulario de contacto se ha diseñado con un fondo de imagen y campos de entrada y botón con estilos uniformes.

## Secciones de la Página

La página web está compuesta por varias secciones, entre las que se incluyen:

- **Información Principal:** Se presenta una descripción de la empresa y sus servicios, resaltando la importancia de la atención médica de emergencia.
- **Equipo Médico:** Se muestra información sobre el equipo médico de la empresa, destacando la experiencia y profesionalismo del personal.
- **Formulario de Contacto:** Se ofrece un formulario de contacto para que los usuarios puedan enviar consultas o solicitar servicios médicos de emergencia.
